# ChibiUltica · 双手巨剑（zweihander）参考贴图

| 文件 | tile id | 用途 |
|------|---------|------|
| `zweihander_item.png` | `zweihander` | 地面 / 物品 32×32 |
| `zweihander_fake_item.png` | `zweihander_fake` | 仿品 |
| `zweihander_inferior_item.png` | `zweihander_inferior` | 劣质版 |
| `overlay_wielded_zweihander.png` | `overlay_wielded_zweihander` | 角色手持 overlay（ChibiNormal.png） |

来源：`incomplete.png` fg 14782–14784（range 14528+）；`ChibiNormal.png` fg 1058（range 224+）。
